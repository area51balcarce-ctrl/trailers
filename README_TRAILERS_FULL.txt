AREA 51 - TRAILERS AUTOMÁTICOS FULL

Esta versión procesa TODO el Excel, no solo 10 juegos.

USO

1. Subí estos archivos a GitHub reemplazando los anteriores.
2. Vercel actualiza automáticamente.
3. Abrí la web.
4. Subí el Excel.
5. Tocá "Completar todos los trailers".
6. Se descarga:
   AREA51_Catalogo_TRAILERS_COMPLETO.xlsx

IMPORTANTE

- No usa API de YouTube.
- Busca dentro del canal @GameTrailers.
- Toma el primer resultado.
- Tiene pausa de seguridad entre búsquedas para no saturar YouTube.
- Si algún juego falla, lo deja vacío y lo muestra en la tabla final.
